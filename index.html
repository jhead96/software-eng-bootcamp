Befunge-93 Interpreter & Visualiser

Jake Head

Befunge-93 is an esoteric programming language created by Chris Presser in 1993. Its most notable feature is the 2D program counter. Unlike tradition programming languages, the program counter is free to move in the cardinals directions on a 2D grid. Programs are written in an 80 x 25 grid and are terminated when an @ character is reached. If the program counter reaches the end of the grid, it will wrap around. It uses a stack (LIFO) for manipulation of data throughout the execution of the program. Turing-complete extensions such as Funge-98 have been subsequently released.

This website implements a Befunge-93 interpreter in JavaScript using a 15x50 grid. It displays the movement of the program counter throughout the grid during execution and the state of the stack and output. It was inspired by a kata from Codewars and was developed for a software bootcamp.
